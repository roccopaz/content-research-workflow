# Discord Report Format — Tech Motivates

Use this format when posting Robo and Star workflow outputs into Discord.

Keep posts easy to scan. No markdown tables.

## Channel Recommendation

Recommended posting flow:
- `#📓-workflows` = planning, SOPs, workflow changes, scheduling
- `#🤖robo` = raw research outputs / trend briefs
- `#🌟star` = content packs / weekly slates
- `#🎬-tech-motivates` = final approved content queue and production decisions

If only one channel is used, keep the post labels very explicit.

## Posting Rules

- Lead with the agent name and report type.
- Keep bullets short and scannable.
- No giant walls of text in the first post.
- If long, use a thread or follow-up reply.
- Keep trend research separate from content packaging.
- Label urgency clearly: Immediate / Weekly / Evergreen.
- If signal quality is weak, say that directly.
